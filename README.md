# New Cwall
全新的Cwall —— 非官方

这是一个基于Vue的Web项目。

## 技术栈
- Vue.js
- TypeScript
- LESS for styling
- Prettier for code formatting

## 安装与运行
1. 确保已安装Node.js和npm
2. 克隆仓库：`git clone https://github.com/BlocksFunction/NewCwall.git`
3. 进入项目目录：`cd NewCwall`
4. 安装依赖：`npm install`
5. 启动开发服务器：`npm run dev`

## 贡献指南
